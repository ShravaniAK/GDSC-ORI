What is web development ?

Web development refers to the creating, building, and maintaining of websites. It includes aspects such as web design, web publishing, web programming, and database management. It is the creation of an application that works over the internet i.e. websites.
But still there is a question remaining to answer is how to make websites and do web development ?

What is Frontend!

  1. The front end is the part of the website users can see and interact with
  2. It is also referred to as the ‘client side of the application.
  3. Front-end web development is the development of the graphical user interface of a website, through the use of HTML, CSS, and JavaScript, so that users can view and interact with that website.

What is Backend!

    It is the part of the website that you cannot see and interact with.Back-end Development refers to the server-side development. It is what goes behind the picture when performing any action on a website. A backend developer needs to work closely with the front-end developers in order to guarantee that the web application's functionality is aligned with the user experience.
